## NCUHOME代码规范 

### 为什么要有团队代码规范？  
虽然这些细节是小事，不会有体验或者性能上的优化，但是却体现了一个coder和团队的专业程度  
团队愿景：成为国内大学卓越的开发团队！  
所以不管团队有多少人，代码风格都应该师出同门！ 

### 感谢
感谢[腾讯AlloyTeam](https://github.com/AlloyTeam/CodeGuide)的代码规范。  
本规范根据团队实际情况，有所改编。
